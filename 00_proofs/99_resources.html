Basic Properties

Note: $\sim$ and $*$ are denoted as an operation
Note: $+$ is denoted as an operation that exhibits associativity
Properties Definition
Reflexivity (Irreflexivity means this property does not hold) $x\sim x$ for all $x$
Symmetry (Asymmetry means this property does not hold) $x\sim y$ and $y\sim x$ for all $x$,$y$
Transitivity (Atransitivity means this property does not hold) $x\sim y$ and $y\sim z$ therefore $x\sim z$ for all $x$,$y$,$z$
Anti-reflexivity $x\sim x$ for no $x$
Anti-Symmetry $x\sim y$ and $y\sim x$ for no $x$,$y$
Anti-Transitivity $x\sim y$ and $y\sim z$ therefore $x\sim z$ for no $x$,$y$,$z$
Associativity $\(x_1+x_2+...+x_n\)+\(x_{n+1}+x_{n+2}+...+x_m\)=x_1+x_2+...+x_m$
Commutativity $x_1+x_2+x_3+...+x_n=x_1+x_3+x_2...+x_n=x_2+x_1+x_3...+x_n=x_2+x_3+x_1...+x_n=...$ (order does not matter)
Distributivity $x*(y\sim z)=x*y\sim x*z$

Basic Elements

Special Elements Definition
Identity (denoted $0$ in this example) $x\sim 0=x$
Inverse (denoted $\neg x$ in this example) $x\sim (\neg x)=0$, Assumes Existence of Identity
